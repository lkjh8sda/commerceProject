# commerceProject

# 1. 개요
다양한 상품을 구매할 수 있는 커머스 프로젝트

# 2. 개발 환경
- Spring boot 2.7.14
- MySql
- JPA
- Redis

- [x] 계정정보 조회
- [x] 상품정보 조회
